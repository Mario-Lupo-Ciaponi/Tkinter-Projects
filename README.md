# Tkinter-Projects
This repository will contain simple tkinter projects.

## Projects:
### Calculator
An simple calculator that can perform the following operations:
- add(e.g. 4 + 5 = 9)
- subtract(e.g. 9 - 5 = 4)
- multiply(e.g. 4 * 5 = 20)
- divide(e.g. 20 / 2 = 10)
- square root(e.g. square root of 49 is 7)
